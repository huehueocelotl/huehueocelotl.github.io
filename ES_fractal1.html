Números complejos

Primero debemos saber lo siguiente de los números complejos:

  • \(z\in \mathbb{C}\), \(z = x + iy\), con \(x,y \in \mathbb{R}\) y \(i^2 = -1\).
  • Su representación en el plano.
  • El módulo y/o longitud y/o valor absoluto \(|z| = \sqrt{x^2 + y^2}\).

Multiplicación de dos números complejos:

La regla general: si \(z = a + ib\) y \(w = x + iy\), entonces \[ z w = (a + ib) ( x + iy) = (ax - by) + i(ay + bx). \]

Funciones complejas

Para entender el fractal de Mandelbrot, es necesarion familiarizarse con la siguente función: para un número complejo \(c = a + ib\) fijo, definimos: \[ f_{c}(z) = z^2 + c. \]

Ejemplo: Tomando \(c = i\), calcular \(f_{c}(1)\) y \(f_{c}(1+i)\).

Iteración

La iteración se refiere a repetir varias veces un proceso. En este caso, iniciando con el valor \(c = 0\), procederemos a calcular \(f_{c}(0)\), \(f_{c}(f_{c}(0))\), \(f_{c}(f_{c}(f_{c}(0)))\), es decir, iteraremos la aplicación de la función \(f_{c}(z)\) evaluada en \(z =0\).

Nota: Es más común usar las expresion \(f^{(3)}_{c}(0)\) para denotar la recurrencia \(f_{c}(f_{c}(f_{c}(0)))\) en estos casos, y de manera análoga definir \(f_{c}(0), f_{c}^{(2)}(0), f_{c}^{(3)}(0), f_{c}^{(4)}(0),\ldots\).

Ejercicios:

  • Para el valor \(c = 1\), calcular \(f_{c}(0), f_{c}^{(2)}(0), f_{c}^{(3)}(0), f_{c}^{(4)}(0),\ldots\).
  • Para el valor \(c = -1\), calcular \(f_{c}(0), f_{c}^{(2)}(0), f_{c}^{(3)}(0), f_{c}^{(4)}(0),\ldots\).
  • Para el valor \(c = 1+i\), calcular \(f_{c}(0), f_{c}^{(2)}(0), f_{c}^{(3)}(0), f_{c}^{(4)}(0),\ldots\).

El conjunto de Mandelbrot y algoritmo de escape

Para cada \(c\in \mathbb{C}\) considere la sucesión: \(z_0 = 0\) y para todo natural \(n\) se define \(z_{n} := f_{c}^{(n)}(0)\).

Si esta sucesión queda acotada, entonces se dice que \(c\) pertenece al conjunto de Mandelbrot, y si no, queda excluido

En la imagen de ejemplo, observamos que el rojo oscuro indica que al cabo de pocos cálculos se sabe que el punto no está en el conjunto mientras que el blanco informa de que se ha tardado mucho más en comprobarlo.

Aproximación:

Como no se puede calcular un sinfín de valores, es preciso poner un límite y decidir que si los \(K\) primeros términos de la sucesión están acotados entonces se considera que el punto pertenece al conjunto. Al aumentar el valor de \(K\) se mejora la precisión de la imagen.

Teorema:
Se sabe que los puntos cuya distancia al origen es superior a 2, es decir, \(|z_{n}|^2 = x^2 + y^2 > 4\), no pertenecen al conjunto. Por lo tanto basta encontrar un solo término de la sucesión que verifique \(|z_{n}| > 2\) para estar seguro de que c no está en el conjunto.
