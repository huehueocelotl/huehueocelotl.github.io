Introducción

En estas notas, aprenderemos a hacer una regresión lineal para ajustar los datos observados en un experimeto.

Nos basaremos en los datos obtenidos en un Laboratorio de química.

El experimento

En el laboratorio de química, el fluorímetro es un aparato que sirve para medir parámetros del espectro visible de fluorescencia.

Estos parámetros son empleados para identificar la cantidad de moléculas específicas en un medio.

Es importante que el aparato se encuentre calibrado para así obtener mediciones más confiables.

En un problema de calibración, queremos a partir de mediciones hechas en muestras estándar, estudiar la relación entre dichas mediciones y el verdadero valor, lo cual nos va a permitir en el futuro, medir una muestra desconocida y tener la certeza de que hemos estimado de manera correcta su verdadero valor.

Los datos

A continuación se describen de manera general, los pasos para obtener los datos.

De manera controlada, se fue modificando la concentración de fluoresceína. Esta sustancia se encuentra por lo general en estdo cristalino.

En cada paso, con el fluorímetro del laboratorio, se fueron tomando las medidas de intensidad de fluorescencia correspondientes.

Las fórmulas

En nestro caso, queremos hacer una regresión linear a partir de los datos experimentales

\begin{equation} (x_1, y_1), (x_2, y_2), \ldots, (x_{n}, y_{n}), \end{equation}

en donde los valores \(x_{i}\)'s corresponden a la variable independiete \(X\), y los valores \(y_{i}\)'s corresponden a la variable dependiente \(Y\).

Recordando la ecuación de la recta

\begin{equation} Y = mX + b, \end{equation}

lo siguiente es estimar los valores the \(m\) y \(b\), de la ecuación de la recta que mejor aproxima nuestros datos experimentales \((x_{i},y_{i})\). La idea es minimizar los errores, los cuales están dados por

\begin{equation} \begin{split} \epsilon_i &= y_i - Y_i \\ &= y_i - mx_i - b.\\ \end{split} \end{equation}

Ahora, definimos a la función \(ERR:\mathbb{R}^2 \to \mathbb{R}\) como

\begin{equation} \begin{split} ERR(m, b)& = \sum_{i=1}^n \epsilon_i^2\\ &=\sum_{i=1}^{n}\left( y_i - mx_i - b \right)^2.\\ \end{split} \end{equation}

La función \(ERR\) es conocida como el error cuadrático.

A partir de nuestros datos experimentales, podemos calcular los siguientes promedios:

\begin{equation} \begin{split} \overline{x} = \frac{1}{n}\sum_{i}x_{i}, \\ \overline{y} = \frac{1}{n}\sum_{i}y_{i}. \end{split} \end{equation}

Entonces, usando de el criterio de las segundas derivadas y resolviendo el sistema de equaciones para \(m\) y \(b\), encontramos que el mínimo valor de ERR se alcanza cuando

\begin{equation} \begin{split} m & = \frac{ \sum_{i=1}^{n} \left(x_i - \bar{x}\right)(y_i - \bar{y}) }{\sum_{i=1}^{n}( x_i - \bar{x})^2},\\ b& = \bar{y} - m\bar{x}. \end{split} \end{equation}

Ahora lo haremos en Python

  • Paso 1. Abre una consola con python3.
  • Paso 2. Carga la librería sklearn.
  • Paso 3. Carga los datos de la variable independiente y la variable dependiente.
  • Paso 4. Ajusta el modelo usando la función LinearRegression.
  • Paso 5. Procede a la interpretación de resultados. Una gráfica siempre es muy útil para este punto.
import numpy as np 
import pandas as pd 
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

df = pd.read_csv('datos.csv') 
# Carga la variable independiente
X = np.array(df['VarX']).reshape(-1, 1) 
# Carga la variable depdendiente
Y = np.array(df['VarY']).reshape(-1, 1) 

# Modelo de regresión linea simple
modelo = LinearRegression()
# Ahora, forma el modelo con los datos
modelo.fit(X, Y)

# Una vez se tiene el modelo, se puede calcular el resultado o hacer predicciones
Y_prediccion = modelo.predict(X)

# Un plot
plt.scatter(X, Y, color ='b') 
plt.plot(X, Y_prediccion, color ='k') 
plt.show()
