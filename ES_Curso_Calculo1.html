Los números reales

Es posible dar una construcción formal de los números reales, sin embargo, ésto sale del propósito de estas notas.

La siguente definición intuitiva será más que suficiente:

Los números reales \(\mathbb{R}\) es el conujunto de todos los números cuya longitud y dirección con respecto al origen \(0\), pueden repersentarse dentro de una línea recta. Éstos números pueden ser de dos tipos:

  • Racionales \(\mathbb{Q}\), es decir, aquellos que pueden representarse por medio de una fracción.
  • Irracionales, aquellos números reales que no son racionales.

Intervalos

Los intervales son subconjuntos especiales de \(\mathbb{R}\):

Considere \(-\infty < a \leq b < \infty\). Se definen los siguentes intervalos

  • \((a,b) = \{ x\in \mathbb{R} : a < x < b \}\).
  • \([a,b] = \{ x\in \mathbb{R} : a \leq x \leq b \}\).
  • \([a,b) = \{ x\in \mathbb{R} : a \leq x < b \}\).
  • \((a,b] = \{ x\in \mathbb{R} : a < x \leq b \}\).
  • \((-\infty,\infty) = \mathbb{R}\).
  • \((-\infty,b) = \{ x\in \mathbb{R} : -\infty < x < b \}\).
  • \((-\infty,b] = \{ x\in \mathbb{R} : -\infty < x \leq b \}\).
  • \((a,\infty) = \{ x\in \mathbb{R} : a < x < \infty \}\).
  • \([a,\infty) = \{ x\in \mathbb{R} : a \leq x < \infty \}\).

Funciones

Una función \(f\) de número reales, denotado por \(f: \mathbb{R} \to \mathbb{R}\), es una regla de correspondencia que asocia de manera única a cada número real \(x\in \mathbb{R}\) con otro número real \(y = f(x) \in \mathbb{R}\). Es decir, si \(f\) es una función y se tiene que \(f(x) = y_1\) y \(f(x) = y_2\) entonces necesariamente \(y_1 = y_2\).

La regla de correspondencia \(f(x) = x^2 + 1\) es una función. Es importante discutir que el hecho de que \(f(1) = f(-1)\) no está en contradicción con la definición.

La siguente regla de correspondencia, no define a una función:

\begin{equation} R(x) = \left\{ \begin{array}{ccc} 2x + 1 & \mbox{para} & x \geq -1\\ -x -2 & \mbox{para} & x < 0 \\ \end{array} \right. \end{equation}

Dominio, Codominio y Rango

Si \(A\subset \mathbb{R}\) y \(f:A\to \mathbb{R}\) satisface la definición de función para todo \(x\in A\), entonces decimos que \(f\) es una función de números reales en \(A\) y

  • \(A\) es llamado el Dominio de \(f\).
  • \(\mathbb{R}\) es llamado el codominio.
  • \(f(A) = \{ f(x) \in \mathbb{R} : x\in A\}\), es llamado el rango de \(f\).

Nótese que \(f(A) \subset \mathbb{R}\) y no siempre \(f(A) = \mathbb{R}\).
