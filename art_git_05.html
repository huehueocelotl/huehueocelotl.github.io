What is a class in python?

In object oriented programming languages, one way to represent real things and real situations is by mean of the so called classes.

Roughly speaking the classes contain:

  • Objects.
  • Properties and behaviour of these objects.
  • And the possibility to create more objects.

By creating an instance it meant the process of creating a new object within a class.

  • One can define actions that can be taken whit the instances
  • One can write a class to extend properties already defined in another existing class.
  • The classes can be stored in Modules.
  • One can also import classes to a Module from other programmers!

How to create a class?

Lets jump into a working example. The class will me myclassroom and each instance Pupil will store a name and an age. Every pupil will have the ability to say their name 

class MyClassroom:
   """ An example of a Class"""
   def __init__(self, name, age):
       self.name = name
       self.age = age

   def say_yourname(self):
       print(f'My name is {self.name} and I am in this classroom')

pupil_1 = MyClassroom('Huehueocelotl',1000)

print(f'The age of pupil_1 is {pupil_1.age} years old.')

pupil_1.say_yourname()

The age of pupil_1 is 1000 years old.
My name is Huehueocelotl and I am in this classroom

As you can see we could access an attribute of our pupul1 using the dot notation. Since every pupil in the classroom has the capacity to say their own name, we only had to call this by writing pupil_1.say_yourname().
